# 商店管理系统-订单管理

- Spring Framework
- Spring MVC
- MyBatis
- JSP
- MySQL
- Log4j

## 如何启动

1. 使用 maven 的打包命令 `mvn package` 可以在 `target` 目录生成 `momas-smbms-1.0.war` 文件
2. 把 `momas-smbms-1.0.war` 复制到 Apache Tomcat 的 webapp 目录下部署
3. 访问 http://localhost:8080/momas-smbms-1.0
4. 默认 用户名 `系统管理员` 密码 `1234567`
